Student Performance Prediction

About the Project

This project is about predicting how well a student will perform in academics based on different factors like study time, attendance, and support from family. The idea is to understand what really affects student performance and use machine learning to make predictions.

Dataset

The dataset includes details such as:

- Age
- Gender
- Study time per week
- Number of absences
- Parental education and support
- Participation in activities
- GPA / Grade (target)

Tools Used

- Python
- Pandas
- NumPy
- Scikit-learn

Model

I used a machine learning model (like Linear Regression / Classification) to train on the data and predict student performance.

What I Did

- Cleaned and prepared the data
- Selected important features
- Trained the model
- Tested its performance
- Made predictions

Result

The model is able to predict student performance (GPA/grade) based on the given inputs. It also helps in understanding which factors influence performance the most.

Example

For example, if a student studies regularly, has fewer absences, and gets good support from parents, the model predicts a higher GPA.
